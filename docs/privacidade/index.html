Política de Privacidade

Última atualização: 22 de maio de 2026

Esta Política de Privacidade governa o uso do aplicativo NurseGuard, desenvolvido para dispositivos móveis. O aplicativo foi projetado com foco total na privacidade, servindo exclusivamente como um sistema local de alerta para administração de medicamentos.

1. Coleta e Uso de Informações Pessoais

O NurseGuard adota uma postura de minimização de dados:

  • Não coleta, não armazena e não transmite qualquer tipo de informação de identificação pessoal, tais como nome, e-mail, telefone ou arquivos do dispositivo.
  • Toda a lógica de funcionamento e armazenamento de dados ocorre de forma estritamente local e criptografada no seu dispositivo.

2. Funcionamento Off-line

  • O aplicativo funciona de forma totalmente off-line.
  • Não solicitamos acesso a recursos sensíveis como GPS, microfone ou contatos. As permissões solicitadas visam estritamente o disparo de notificações e alarmes precisos.

3. Serviços de Terceiros e Anúncios

Para fins de sustentabilidade, o aplicativo poderá integrar o serviço Google AdMob.

  • O AdMob pode utilizar identificadores anônimos do dispositivo para exibir anúncios relevantes.
  • Nenhum dado pessoal é compartilhado com esta plataforma, visto que o aplicativo não os coleta.
  • Consulte a política do provedor aqui: Google AdMob Privacy Policy.

4. Segurança dos Dados

Como não realizamos upload para nuvem, seus dados estão protegidos contra interceptação de rede. O armazenamento local é protegido pelo sistema de chaves do próprio Android (Keystore).

5. Alterações nesta Política

Esta página poderá ser atualizada periodicamente. Recomendamos a revisão eventual para verificar modificações, que entram em vigor imediatamente após a publicação.

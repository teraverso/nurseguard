Política de Privacidade

Última atualização: 22 de maio de 2026

Esta Política de Privacidade governa o uso do aplicativo NurseGuard, desenvolvido para dispositivos móveis. O aplicativo foi projetado com foco total na privacidade, servindo exclusivamente como um sistema local de alerta para administração de medicamentos.

01. Coleta e Uso de Informações

O NurseGuard adota uma postura técnica de minimização de dados:

  • Não Coleta: O aplicativo não coleta, não armazena e não transmite qualquer tipo de informação de identificação pessoal (PII), tais como nome, e-mail, telefone ou arquivos do dispositivo.
  • Processamento Local: Toda a lógica de funcionamento e armazenamento de dados ocorre de forma estritamente local e criptografada no seu dispositivo.

02. Funcionamento Off-line

  • O aplicativo funciona de forma totalmente off-line, garantindo autonomia técnica.
  • Não solicitamos acesso a recursos sensíveis como GPS, microfone ou contatos. As permissões visam estritamente o disparo de notificações precisas.

03. Serviços de Terceiros

Para fins de sustentabilidade, o aplicativo integra o serviço Google AdMob para exibição de anúncios:

  • O AdMob pode utilizar identificadores anônimos do dispositivo para exibir anúncios relevantes.
  • Nenhum dado pessoal ou de saúde é compartilhado com esta plataforma.
  • Consulte a política do provedor aqui: Google AdMob Privacy Policy.

04. Segurança dos Dados

Inexistência de upload para nuvem, protegendo os dados contra interceptação de rede. O armazenamento local é protegido pelo sistema de chaves do próprio Android (Keystore).

05. Alterações e Contato

Esta política poderá ser atualizada periodicamente. Recomendamos a revisão eventual para verificar modificações que entram em vigor imediatamente após a publicação.

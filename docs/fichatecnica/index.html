Ficha Técnica NurseGuard

Solução em HealthTech para adesão terapêutica e design inclusivo para idosos, cuidadores e neurodivergentes.

01. Visão Geral

O NurseGuard é um sistema gerenciador de medicamentos projetado para mitigar a baixa adesão terapêutica. A solução integra engenharia de software de alta performance com design cognitivo, focando na resiliência do sistema e na experiência do usuário final.

02. Público e UX

A arquitetura de interface foi desenvolvida sob os critérios de acessibilidade WCAG, com foco em:

  • Gerontodesign: Interfaces para idosos com zonas de toque expandidas e alto contraste.
  • Neurodiversidade: Fluxos simplificados para usuários com TEA e TDAH, reduzindo a carga cognitiva e estímulos desnecessários.
  • Multi-Perfil: Separação lógica entre perfis de Paciente e Cuidador.

03. Engenharia de Software

Arquitetura

O projeto segue o padrão MVVM (Model-View-ViewModel), garantindo separação de interesses e testabilidade. A interface é 100% declarativa, utilizando o framework Jetpack Compose.

Sistemas de Segundo Plano

Utilização de AlarmManager de precisão e WorkManager para garantir o disparo de notificações críticas, contornando as otimizações de bateria agressivas do ecossistema Android.

04. Segurança e Privacidade

O sistema foi construído sob o princípio de Privacy by Design:

  • Armazenamento Local: Dados de saúde permanecem exclusivamente no hardware do usuário.
  • Zero-Cloud: Inexistência de backend para armazenamento de dados sensíveis, eliminando vetores de ataque em nuvem.
  • Criptografia: Proteção de credenciais de acesso via sistema de chaves do sistema operacional (Android Keystore).

05. Diferenciais Estratégicos

  • Resiliência Offline: Operação plena sem dependência de conectividade.
  • Gestão de Inventário: Algoritmo de controle de estoque com alertas preditivos (Semáforo Visual).
  • Escalabilidade: Código modularizado que permite a implementação rápida de novas funcionalidades de telemetria de saúde.
